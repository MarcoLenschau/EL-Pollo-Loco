    CollectObject

    Represents a collectable object that is movable and can be drawn on the canvas. This class extends from the MovableObject class and is used for objects that can be collected in a game or app.

    Constructor

    new CollectObject(imgPath)

    Creates an instance of the CollectObject class, loading an image from the provided path and setting a random position. The object will be placed at a random x position between 150 and 2350, and a random y position between 0 and 150. It calls the constructor of the MovableObject class to inherit its functionality.
